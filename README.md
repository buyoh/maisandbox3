# maisandbox3

## setup (for develop)

- Ubuntu 20.04 (recommended. may be runnable on *NIX)
- node v12
- ruby 2.7

install yarn

```
npm i yarn
npx yarn
```

or

```
npm i -g yarn
yarn
```

- - -

install tools

```
yarn
bundle install  # for RSpec only
```

build...

```
yarn build
```

execute

```
yarn dev
```

test

```
bundle exec rspec
yarn test
```
